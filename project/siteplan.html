Color Schema

I will use the main colors of dark green and light gray. I will also use some black for maybe some certain headers with white text. I will use green for the main header and footer content to represent the website color, and use light gray for the background in the body since there's going to be lots of text on my website.

Typography

For my website font, I will use the Roboto Condensed font for everything. Headers, body, main, and footer.

Wireframe

My wireframe will just be basic and become smaller when it's in the mobile view. In the desktop version, I will have some whitespace on the sides so everything is more centered. On the mobile page it will be streched out to the devices width.
